# Job-Helper-Agent
Simple AI agent for job hunting

## What It Does

An AI-powered career assistant that helps users match their
CV/resume against job postings. It can read CV files,
fetch job listings, and provide tailored advice on job
suitability.

- Read and analyze CV/resume files (PDF, DOCX, DOC)
- Fetch and summarize job postings from URLs
- Compare CV against job requirements
- Suggest CV improvements for target roles

## Set up
```
python -m pip install -r requirements.txt
```
